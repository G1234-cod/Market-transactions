## 🔴 P0 — 阻塞性（不解决系统跑不起来）

### 1. API Key 配置（DeepSeek + 阿里云百炼）

**现状**: `.env.example` 模板已写好，但需要申请真实 Key 填入 `.env`。

| Key | 用途 | 申请地址 | 费用 |
|-----|------|----------|------|
| `DASHSCOPE_API_KEY` | Qwen-VL-Max 视觉识别 | [阿里云百炼控制台](https://bailian.console.aliyun.com/) | 按量付费 |
| `DEEPSEEK_API_KEY` | DeepSeek 文案生成 + 瑕疵定价 | [DeepSeek 开放平台](https://platform.deepseek.com/) | 按量付费 |

**配置步骤**:

```bash
# 1. 复制模板
cp backend/.env.example backend/.env

# 2. 编辑 backend/.env，填入真实 Key
DASHSCOPE_API_KEY=sk-xxxxxxxxxxxxxxxx     # 阿里云百炼
DEEPSEEK_API_KEY=sk-xxxxxxxxxxxxxxxx      # DeepSeek 官方

# 3. 验证配置
cd backend
python -c "from app.config import validate_config; validate_config()"
```

**注意事项**:
- 两个 Key 缺一不可——Qwen-VL 负责图片识别，DeepSeek 负责文案生成和定价
- 启动时 `validate_config()` 会自动检查，没配置会打印 ❌ 错误
- 国内访问 DeepSeek 不需要代理，阿里云百炼也不需要
