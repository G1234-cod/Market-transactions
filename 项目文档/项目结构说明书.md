### 3.2 应用入口 —— main.py

### 3.3 认证模块 —— app/auth/jwt_handler.py

| 功能 | 实现 |
|------|------|
| Token 创建 | JWT HS256，sub=user_id, exp=7天 |
| Token 验证 | 中间件 decode + 数据库二次确认 |
| 密码哈希 | PBKDF2-SHA256, 16 字节随机盐, 10 万次迭代 |
| 用户提取 | `dependencies.py` 提供 FastAPI `Depends(get_current_user)` |

### 3.4 常量定义 —— app/constants/categories.py

预定义品类品牌映射字典，用于：
- 前端品类下拉选择
- 模型识别结果归一化
- 搜索过滤选项

### 3.6 业务服务详细说明

| 服务文件 | 职责 | 关键设计 |
|---------|------|---------|
| `audit_service.py` | LLM 调用审计记录 | 幂等设计（查询去重），API 调用鉴证 |
| `text_service.py` | 流式文案生成 | SSE 通道管理，前端中断支持 |
| `price_service.py` | 市场查价 | 三层匹配：精确（品牌+型号）→ 模糊（单品牌）→ 空（全品类均价） |
| `vision_service.py` | Qwen-VL 响应解析 | JSON 容错提取，置信度归一化 |
| `damage_detector.py` | 瑕疵分级 | 4 级分级（极轻/轻微/中等/严重）+ 7 类中文标签 |
| `image_processor.py` | 图片处理 | 缩放/裁剪/格式转换，为模型准备输入 |
| `crawler_service.py` | 价格爬取 | 多源价格采集调度 |
| `history_estimator.py` | AI 价格估算 | 基于历史数据 + LLM 推理估算合理价格 |

### 3.7 LLM 客户端架构

```
LLMBase (base.py, 抽象基类)
├── DeepSeekClient (deepseek_client.py)
│   └── chat_stream()      → SSE 流式文本生成
├── DeepSeekPriceClient (deepseek_price_client.py)
│   └── estimate_price()   → 智能定价 + 扣款明细
└── QwenVLClient (qwen_vl_client.py)
    └── recognize()         → 视觉识别（双通道容错：OpenAI兼容 → DashScope原生）
```

**设计模式**：策略模式（Strategy Pattern），通过抽象基类统一接口，新增模型只需创建子类实现。
