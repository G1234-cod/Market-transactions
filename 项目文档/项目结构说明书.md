# 智能二手商品发布助手 —— 项目文档

## 一、项目概述

面向二手商品交易的一站式 AI 发布助手。上传商品图片 → AI 视觉识别品类/品牌/型号/成色 → 自动查询市场行情 → DeepSeek 流式生成带货文案 → 一键发布到商城。支持瑕疵检测、AI 智能定价、以图搜图。


### 技术栈

| 层级 | 技术 | 说明 |
|:---|:---|:---|
| **后端框架** | Python + FastAPI | 原生 async/await，支持 SSE 流式 |
| **数据库** | MySQL + aiomysql | 异步连接池，8 张业务表 |
| **向量数据库** | Qdrant | Docker 部署，HNSW 索引 |
| **前端** | Vue 3 + Vite + TailwindCSS | 组合式 API，响应式设计 |
| **HTTP 客户端** | Axios | 拦截器统一认证 |
| **目标检测** | YOLOv8 | 物品检测 + 瑕疵检测 |
| **多模态** | CLIP | 以图搜图特征提取 |
| **视觉识别** | Qwen-VL-Max | 商品结构化识别 |
| **文本生成** | DeepSeek | 文案生成 + 智能定价 |
| **图像处理** | rembg + OpenCV | 去背景 + 去噪 |

## 六、核心数据流

### 6.1 完整发布流程

```text
用户上传图片
    │
    ▼
POST /api/v1/extract
(Qwen-VL 识别: category + brand + model + condition)
    │
    ▼
GET /api/v1/price
(市场行情: avg_price + low_price + high_price)
    │
    ▼
POST /api/v1/generate
(DeepSeek SSE 流式生成标题 + 描述)
    │
    ▼
用户确认/编辑
    │
    ▼
POST /api/v1/history/save
(写入 MySQL + 自动加入 Qdrant 索引)
```

### 6.2 瑕疵检测 + AI 定价流程

```text
用户上传图片 + 品类/品牌/型号
    │
    ▼
POST /api/v1/process/image
    │
    ▼
预处理管道
├── 尺寸统一 (448×448)
├── 去背景 (rembg)
└── 去噪 (OpenCV 高斯滤波)
    │
    ▼
YOLO 瑕疵检测
├── scratch (划痕)
├── dent (磕碰)
├── crack (裂痕)
├── stain (污渍)
└── peeling (掉漆)
    │
    ▼
画框标注
├── 圆形 + 红色 = 重度
├── 矩形 + 橙色 = 中度
├── 多边形 + 黄色 = 轻度
└── 虚线框 + 蓝色 = 轻微
    │
    ▼
查询市场均价 + DeepSeek 瑕疵定价
├── 折扣比例
├── 建议售价
└── 定价理由
```

### 6.3 以图搜图流程

```text
用户上传图片 / 输入文字
    │
    ▼
CLIP 提取特征向量 (512维)
    │
    ▼
Qdrant 向量检索 (COSINE 相似度)
    │
    ▼
返回 Top-K 商品ID
    │
    ▼
MySQL 查询商品详情
    │
    ▼
返回相似商品列表 + 相似度分数
```
