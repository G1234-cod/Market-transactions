
# 智能二手商品发布助手 —— 项目结构说明书


## 一、项目概述

| 项目名称 | 智能二手商品发布助手 |
|:---|:---|
| **版本** | 1.0.0 |
| **技术栈** | FastAPI + Vue 3 + YOLOv8 + CLIP + Qwen + DeepSeek + MySQL + Qdrant |
| **项目类型** | AI 驱动的全栈 Web 应用 |
| **核心功能** | 图片识别、瑕疵检测、以图搜图、智能定价、一键发布 |

## 五、数据库表结构

| 表名 | 用途 | 核心字段 |
|:---|:---|:---|

## 七、核心调用链路

### 7.1 商品发布流程

```
用户上传图片
    ↓
POST /api/v1/extract（Qwen 识别）
    ↓
POST /api/v1/process/image（瑕疵检测 + 定价）
    ↓
前端自动填表
    ↓
POST /api/v1/history/save（保存商品）
    ↓
商品出现在商城
```

### 7.2 以图搜图流程

```
用户上传图片
    ↓
POST /api/v1/search/image
    ↓
CLIP 提取特征向量
    ↓
Qdrant 向量检索
    ↓
查询 MySQL 商品详情
    ↓
返回相似商品列表
```
