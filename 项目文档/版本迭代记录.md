## 📝 详细迭代记录

### v0.1 —— 项目初始化（2026-06-22 上午）

**技术选型：** FastAPI + Vue 3 + YOLOv8 + CLIP + Qwen-VL-Max + DeepSeek + MySQL 8.0 + Qdrant

**产出：** 需求文档、项目目录结构、README、.gitignore

**里程碑：** 项目正式启动

---

### v0.2 —— 数据库 + 后端骨架 + 用户系统（2026-06-22 下午）

**数据库：** 8 张表（users, market_prices, published_items, ai_audit_logs, item_likes, hard_cases, feature_vectors, price_history）、aiomysql 异步连接池

**用户系统：** PBKDF2-SHA256 密码哈希（100,000 迭代）、注册/登录

**新增文件：** `connection.py`、`crud.py`、`schema.sql`、`seed.sql`、`auth.py`、`jwt_handler.py`、`schemas.py`

**里程碑：** 数据库 + 用户认证就绪

---

### v0.3 —— AI 识别 + YOLO 检测（2026-06-23 上午）

**Qwen-VL-Max：** 视觉识别集成、结构化 JSON 输出、图片上传保存

**YOLOv8：** yolov8n.pt 预训练模型、80 类 COCO 检测、多色画框标注、Base64 输出

**API：** `/register`、`/login`、`/extract`、`/yolo/detect`、`/yolo/health`

**里程碑：** AI 双模型基础链路打通

---

### v0.4 —— 瑕疵检测 + 程度分级（2026-06-23 下午）

**瑕疵类别：** 5 类（划痕/磕碰/污渍/裂痕/其他）

**程度分级（视觉编码）：**

|形状|颜色|程度|
|---|---|---|
|圆形|红色 #FF0000|重度|
|矩形|橙色 #FF8C00|中度|
|多边形|黄色 #FFD700|轻度|
|虚线框|蓝色 #00BFFF|轻微|

**新增：** `defect_detector_yolo.py`、`process.py`、`image_utils.py`

**里程碑：** 瑕疵检测 + 分级标注完成

---

### v0.5 —— 图像预处理 + 双模型比对（2026-06-24 上午）

**预处理管道：** 448×448 尺寸统一 → rembg 去背景 → OpenCV 去噪 → 多物体判断

**双模型比对：** YOLO + Qwen 并行识别 → 结果比对 → 不一致数据自动收集

**新增：** `preprocess.py`、`data_collector.py`、`font_utils.py`、`file_validator.py`

**里程碑：** 数据飞轮第一步 — 错误收集启动

---

### v0.6 —— DeepSeek 定价 + 发布系统（2026-06-24 下午）

**定价：** 瑕疵程度 + 市场行情 → DeepSeek 智能建议价、三层查价匹配（精确→模糊→空）

**发布：** 商品保存、商城列表、下架/上架、SSE 流式文案生成（打字机效果）

**API：** `/price`、`/history`、`/history/save`、`/history/{id}/delist`、`/history/{id}/publish`、`/market`、`/generate`

**里程碑：** 识别→检测→定价→保存 全链路打通

---

### v0.7 —— 前端 + 以图搜图（2026-06-25 上午）

**前端 6 页面：** 登录/注册/发布/商城/历史/详情、Vue 3 + Pinia + TailwindCSS + 响应式

**以图搜图：** CLIP (ViT-B-32) 特征提取 → Qdrant 向量检索 (COSINE, 512维) → 以图搜图 + 以文搜图

**API：** `/search/image`、`/search/text`、`/search/index`、`/search/stats`、`/search/health`

**里程碑：** 前端 + 搜索完整上线
