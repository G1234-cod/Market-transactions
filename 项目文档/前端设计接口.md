# 智能二手商品发布助手 —— 接口设计文档（前端版）


## 📋 接口概览

| 模块 | 接口数量 | 说明 |
|:---|:---:|:---|
| 用户 | 2 个 | 注册/登录 |
| 图片识别 | 1 个 | Qwen 视觉识别 |
| YOLO 检测 | 1 个 | 物品检测 |
| 瑕疵检测 | 1 个 | 全链路图片处理 |
| 以图搜图 | 2 个 | 图搜图/文搜图 |
| 价格 | 1 个 | 行情查询 |
| 发布历史 | 2 个 | 列表/保存 |
| 商品操作 | 2 个 | 下架/上架 |
| 商城 | 1 个 | 商品列表 |
| **总计** | **13 个** | |

| 项目 | 说明 |
|:---|:---|

## 十、前端 API 封装（JavaScript）

```javascript
// frontend/src/api/index.js

import axios from 'axios'

const api = axios.create({
  baseURL: '/api/v1',
  timeout: 30000,
})

// ============================================================
// 用户
// ============================================================
export const register = (username, password) =>
  api.post('/register', { username, password }).then(res => res.data)

export const login = (username, password) =>
  api.post('/login', { username, password }).then(res => res.data)

// ============================================================
// 图片识别
// ============================================================
export const extractImage = (file, userId = 1) => {
  const form = new FormData()
  form.append('image', file)
  form.append('user_id', String(userId))
  return api.post('/extract', form).then(res => res.data)
}

// ============================================================
// YOLO 检测
// ============================================================
export const yoloDetect = (file) => {
  const form = new FormData()
  form.append('image', file)
  return api.post('/yolo/detect', form).then(res => res.data)
}

// ============================================================
// 瑕疵检测
// ============================================================
export const processImage = (file, params = {}) => {
  const form = new FormData()
  form.append('image', file)
  if (params.user_id) form.append('user_id', String(params.user_id))
  if (params.item_id) form.append('item_id', String(params.item_id))
  if (params.category) form.append('category', params.category)
  if (params.brand) form.append('brand', params.brand)
  if (params.model) form.append('model', params.model)
  if (params.market_avg_price) form.append('market_avg_price', String(params.market_avg_price))
  return api.post('/process/image', form).then(res => res.data)
}

// ============================================================
// 以图搜图
// ============================================================
export const searchByImage = (file, topK = 10) => {
  const form = new FormData()
  form.append('image', file)
  form.append('top_k', String(topK))
  return api.post('/search/image', form).then(res => res.data)
}

export const searchByText = (text, topK = 10, category = '') => {
  const form = new FormData()
  form.append('text', text)
  form.append('top_k', String(topK))
  if (category) form.append('category', category)
  return api.post('/search/text', form).then(res => res.data)
}

// ============================================================
// 价格查询
// ============================================================
export const queryPrice = (brand, model) =>
  api.get('/price', { params: { brand, model } }).then(res => res.data)

// ============================================================
// 发布历史
// ============================================================
export const getHistory = (userId, params = {}) =>
  api.get('/history', { params: { user_id: userId, ...params } }).then(res => res.data)

export const saveHistory = (data) =>
  api.post('/history/save', data).then(res => res.data)

// ============================================================
// 商品操作
// ============================================================
export const delistItem = (itemId) =>
  api.post(`/history/${itemId}/delist`).then(res => res.data)

export const publishItem = (itemId) =>
  api.post(`/history/${itemId}/publish`).then(res => res.data)

// ============================================================
// 商城
// ============================================================
export const getMarket = (params = {}) =>
  api.get('/market', { params }).then(res => res.data)

// ============================================================
// 工具函数
// ============================================================
export const base64ToImage = (base64) => {
  return `data:image/png;base64,${base64}`
}
```


## 十一、前端使用示例

```vue
<template>
  <div>
    <!-- 上传图片 -->
    <input type="file" @change="handleUpload" accept="image/*" />
    
    <!-- 展示结果 -->
    <div v-if="result">
      <h3>识别结果</h3>
      <p>品类：{{ result.data?.category }}</p>
      <p>品牌：{{ result.data?.brand }}</p>
      <p>型号：{{ result.data?.model }}</p>
      <p>瑕疵数量：{{ result.defect_count }}</p>
      <img :src="base64ToImage(result.annotated_base64)" />
    </div>
  </div>
</template>

<script setup>
import { ref } from 'vue'
import { processImage, base64ToImage } from '@/api'

const result = ref(null)

const handleUpload = async (event) => {
  const file = event.target.files[0]
  if (!file) return
  
  try {
    const res = await processImage(file, {
      user_id: 1,
      category: '手机',
      brand: 'Apple',
      model: 'iPhone 14 Pro'
    })
    result.value = res.data
  } catch (error) {
    console.error('上传失败:', error)
  }
}
</script>
```

## ⚠️ 前端开发注意事项


### 一、CORS 跨域问题

你的后端已经配置了 CORS，允许所有来源访问。但开发时如果遇到跨域报错，检查一下：

```python
# backend/main.py —— 你已经有了
app.add_middleware(
    CORSMiddleware,
    allow_origins=["*"],              # ✅ 允许所有来源
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)
```

> **注意：** 如果前端用 `localhost:5173`，后端是 `localhost:8000`，需要确保代理配置正确。

**Vite 代理配置（`frontend/vite.config.js`）：**
```javascript
export default {
  server: {
    proxy: {
      '/api': {
        target: 'http://localhost:8000',
        changeOrigin: true,
      },
      '/static': {
        target: 'http://localhost:8000',
        changeOrigin: true,
      }
    }
  }
}
```


### 二、图片上传格式

| 注意事项 | 说明 |
|:---|:---|
| **FormData** | 图片必须用 `FormData` 发送 |
| **Content-Type** | 不要手动设置，浏览器自动处理 |
| **文件大小** | 建议限制 10MB 以内（后端也有校验） |
| **图片格式** | 支持 JPG/PNG/WebP |

```javascript
// ✅ 正确
const formData = new FormData()
formData.append('image', file)
await axios.post('/api/v1/process/image', formData)

// ❌ 错误（不要用 JSON）
await axios.post('/api/v1/process/image', { image: file })
```


### 三、Base64 图片展示

你的接口返回的是 **Base64 字符串**（`annotated_base64`、`bg_removed_base64`），前端需要这样展示：

```vue
<template>
  <!-- ✅ 直接用 data:image 协议 -->
  <img :src="`data:image/png;base64,${imageData}`" />
</template>

<script setup>
// imageData 是接口返回的 base64 字符串（不含 data:image 头）
const imageData = result.annotated_base64
</script>
```


### 四、SSE 流式接收（文案生成）

`/generate` 接口是 SSE 流式输出，不是普通 JSON 接口。

```javascript
// ✅ 正确（用 EventSource 或 fetch）
const eventSource = new EventSource('/api/v1/generate', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify(payload)
})

eventSource.onmessage = (event) => {
  const data = JSON.parse(event.data)
  if (data.content) {
    // 逐字显示
    text += data.content
  }
  if (data.done) {
    // 生成完成
    eventSource.close()
  }
}
```


### 五、错误处理规范

你的后端错误响应格式是：

```json
{
  "detail": "错误信息"   // FastAPI 默认格式
}
```

前端统一拦截：

```javascript
// axios 拦截器
api.interceptors.response.use(
  response => response,
  error => {
    const message = error.response?.data?.detail || '请求失败，请重试'
    // 显示错误提示
    showToast(message, 'error')
    return Promise.reject(error)
  }
)
```

    user: null,
    token: null,
  }),
  actions: {
    async login(username, password) {
      const res = await api.login(username, password)
      this.user = res
      localStorage.setItem('user', JSON.stringify(res))
    },
    logout() {
      this.user = null
      localStorage.removeItem('user')
    }
  }
})
