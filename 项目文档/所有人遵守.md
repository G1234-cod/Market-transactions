## 🐍 第二步：安装 Python

1. 浏览器访问 [python.org](https://www.python.org)
2. 点击 **Downloads** → **Python 3.10.x**（推荐 3.10.11）
3. 下载后双击安装
4. ⚠️ **务必勾选 "Add Python to PATH"**
5. 点击 **Install Now**

**验证安装**（在 CMD 中执行）：
```cmd
python --version
```
预期输出：`Python 3.10.x`


## 🗄️ 第三步：安装 MySQL

### Windows 安装：

1. 浏览器访问 [dev.mysql.com/downloads/installer](https://dev.mysql.com/downloads/installer/)
2. 下载 **MySQL Installer 8.0.xx**
3. 双击运行，选择 **Developer Default**
4. 一路 Next，设置 **root 密码**（⚠️ 务必记住这个密码！）
5. 完成安装

**验证安装**：
```cmd
mysql --version
```

## 🚀 第九步：训练通用识别模型（YOLOv8s on COCO）

```cmd
python train.py
```

**训练参数说明**（如需修改，用记事本打开 `train.py`）：
- `BATCH = 8` → 若显存 < 6GB，改为 `BATCH = 4`
- `EPOCHS = 5` （每次运行训练 5 轮）

**训练机制**：
- 首次运行从 COCO 预训练权重开始
- 后续运行会自动从断点恢复（`last.pt`）
- 连续 10 轮 mAP 无提升则自动停止

**训练完成后模型位置**：
```
runs/train/coco_yolov8s/weights/best.pt
```

## ⚠️ 常见问题及解决

| 问题 | 解决办法 |
|:---|:---|
| CUDA out of memory | 调小 `BATCH`（如 8→4 或 2） |
| pip 下载慢 | 使用清华源：`pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple` |
| 训练中断 | 重新运行 `python train.py`，自动从 `last.pt` 断点继续 |
| MySQL 连不上 | 检查服务是否启动：`net start MySQL80` |
| 找不到模块 | 确认虚拟环境已激活：`venv\Scripts\activate` |
| torch 不是 GPU 版 | 重新运行 `torchzidong.py`，或手动安装：`pip install torch torchvision --index-url https://download.pytorch.org/whl/cu118` |
