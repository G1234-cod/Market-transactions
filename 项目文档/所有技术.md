## 四、部署架构

```
┌─────────────────────────────────────────────────────────────────┐
│                         用户浏览器                              │
│                    (Vue 3 SPA 应用)                            │
└────────────────────────┬────────────────────────────────────────┘
                         │ HTTPS
                         ▼
┌─────────────────────────────────────────────────────────────────┐
│                        Nginx (反向代理)                         │
│              / → 前端静态文件  /api → 后端服务                   │
└────────────────────────┬────────────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────────────┐
│                     FastAPI Server                              │
│                  (Uvicorn ASGI)                                 │
└──────────┬──────────────┬──────────────┬───────────────────────┘
           │              │              │
           ▼              ▼              ▼
┌─────────────────┐ ┌─────────────┐ ┌─────────────────────────┐
│     MySQL       │ │   Qdrant    │ │   AI API 服务            │
│  (业务数据)      │ │  (向量检索)  │ │  (Qwen / DeepSeek)      │
└─────────────────┘ └─────────────┘ └─────────────────────────┘
```

## 七、技术亮点总结

| 类别 | 技术点 | 说明 |
|:---|:---|:---|
| **异步架构** | FastAPI + aiomysql | 全链路异步，高并发支持 |
| **流式输出** | SSE (Server-Sent Events) | 打字机效果文案生成 |
| **多模态检索** | CLIP + Qdrant | 以图搜图/以文搜图 |
| **双模型验证** | YOLO + Qwen | 并行识别，交叉验证 |
| **数据飞轮** | 错误收集 + 自动训练 | 系统自进化能力 |
| **策略模式** | LLM 抽象基类 | 模型无缝切换 |
| **向量检索** | HNSW 索引 | 毫秒级相似度搜索 |
| **容器化** | Docker | 一键部署 Qdrant |
