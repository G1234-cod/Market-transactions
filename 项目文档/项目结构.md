## 后端核心数据流（路由 → 服务 → 数据库/LLM）

```
前端请求
    │
    ▼
routers/auth.py             ──►  db/crud.py
    │  POST /api/v1/login          register_user / authenticate_user (PBKDF2)
    │  POST /api/v1/register
    │
    ▼
routers/extract.py          ──►  services/vision_service.py   ──►  llm/qwen_vl_client.py
    │  POST /api/v1/extract         调 Qwen-VL-Max 识别图片           阿里云百炼 API（Base64 Data URI）
    │
    ▼
routers/price.py            ──►  services/price_service.py    ──►  db/crud.py
    │  GET /api/v1/price           三层匹配查询行情                  MySQL market_prices 表
    │
    ▼
routers/generate.py         ──►  services/text_service.py     ──►  llm/deepseek_client.py
    │  POST /api/v1/generate       拼 Prompt → 流式生成              DeepSeek 官方 API
    │  (SSE StreamingResponse)
    │
    ▼
routers/history.py          ──►  db/crud.py
    │  GET/POST /api/v1/history/*   查询/保存/下架/发布                MySQL published_items 表
    │
    ▼
routers/market.py           ──►  db/crud.py
    │  GET /api/v1/market          JOIN users 商城列表                MySQL published_items + users
    │
    ▼
services/audit_service.py   ──►  db/crud.py
  异步写审计日志                  INSERT ai_audit_logs
```

## 关键设计决策

| 文件/模块 | 设计意图 |
|-----------|---------|
| `llm/base.py` 抽象类 | 策略模式：未来切换模型只需新增子类，不改业务代码 |
| `llm/qwen_vl_client.py` | 双接口支持：OpenAI 兼容 + DashScope 原生 SDK，含 `build_vision_message` 等工具方法 |
| `llm/deepseek_client.py` | 基于 AsyncOpenAI 的流式客户端，与 `base.py` 解耦 |
| `crud.py` PBKDF2 密码哈希 | SHA256 + 100,000 迭代 + 16 字节随机 salt，无法反推明文，零外部依赖 |
| `store/user.js` | Vue 3 composable 全局用户状态 + localStorage 持久化，刷新不掉线 |
| `main.js` 路由守卫 | `meta.auth` 拦截未登录、`meta.guest` 拦截已登录、`meta.public` 无需登录 |
| `ConfirmCard.vue` 三级校验 | ok（绿框）/ warn（黄框提示但放行）/ error（红框拦截），兼顾安全与罕见品牌兼容性 |
| `services/` 层 | 隔离路由与数据库：路由只管 HTTP，业务逻辑集中在 service |
| `models/schemas.py` | Pydantic 模型：统一前后端数据结构校验，避免散落各处的 dict |
| `api/index.js` 中的 SSE 工具 | 基于 `fetch` + `ReadableStream` 而非 EventSource（EventSource 不支持 POST） |
| `vision_service.py` 的双重校验 | Prompt 约束 + `json.loads` + Pydantic 校验：三层防护模型 JSON 输出异常 |
| `price_service.py` 的三层匹配 | 精确→difflib→空值：覆盖 AI 识别结果与数据库命名不一致的边界情况 |
| `static/uploads/` 本地存储 | 开发阶段最简方案，FastAPI `StaticFiles` 直接挂载即可访问 |
| `App.vue` Toast 通知系统 | provide/inject 全局通知：success/error/warning/info 四种类型，自动消失 |
| `vue <transition>` 动画 | 页面切换 + 步骤切换均有渐变/位移动画，流畅引导用户体验 |
| `extract.py` Base64 传图 | 图片转 Data URI 直传 Qwen-VL，无需外部可访问 URL |
